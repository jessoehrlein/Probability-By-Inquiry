4.2 Definitions, Exercises, and Theorems

Definition 4.1: Random Variable

A random variable \(X\) is a function from the sample space to the real numbers. That is, \(X: S \rightarrow \mathbb{R}\).

Here are some examples of random variables: the sum of rolls of two dice, the number of heads in a sequence of 8 coin flips, the total winnings from playing a particular probability-based game, the number of hats given to the correct people when \(n\) people are handed their hats back at random.

Exercise 4.1 Find the possible values (ranges of the functions) for each of these random variables:

  1. The number of heads in three coin flips

  2. The time until you next receive an email

  3. The number of flips of a coin until either the first heads appears or \(n\) flips occur

  4. The number of flips of a coin until the first heads appears

  5. The time until a cell phone is either plugged into a charger or dies

  6. The number of numbers 7 or above when four numbers from 0-9 are selected randomly, with repeats allowed

  7. The number of numbers 7 or above when four numbers from 0-9 are selected randomly, with repeats not allowed

  8. The time until the next occurrence of rain in a particular city

Exercise 4.2 Write the function definition for the random variable in Exercise 4.1 (a). In other words, find the sample space and map each outcome to a real number based on this random variable.

We’ll separate random variables into multiple types, and we’ll talk about two of those types: discrete and continuous.

Definition 4.2: Discrete Random Variable

A discrete random variable is a random variable with a countable range. That is, it can only take on countably many values.

We’ll formally define a continuous random variable later, but for the moment, consider them to be random variables with uncountably infinite ranges.

Exercise 4.3 Which random variables from Exercise 4.1 are discrete?

Okay, so we have discrete random variables. How do we start talking about probabilities associated with them?

Definition 4.3: Probability Mass Function

Let \(X\) be a discrete random variable with range \(\{x_1, x_2, x_3, \cdots\}\) The probability mass function (or PMF) \(P_X\) assigns a probability to each value in the range. That is, \(P_X(x_k) = P(X = x_k)\) for every \(x_k\) in \(X\)’s range.

Exercise 4.4 Find the probability mass function for the random variable you defined in Exercise 4.1 (a) and Exercise 4.2.

Exercise 4.5 I have a fair six-sided die which I roll until I roll a 6 for the first time. Let \(X\) be the total number of die rolls. Find the probability mass function of \(X\).

The PMF isn’t the only useful way to talk about probabilities associated with a discrete random variable.

Definition 4.4: Cumulative Distribution Function

The cumulative distribution function (or CDF) of a random variable \(X\) is \(F_X (x) = P(X \leq x)\) for all \(x \in \mathbb{R}\).

It’s often useful to think about the “typical” or value of a random variable. Let’s define that formally.

Definition 4.5: Expected Value

The expectation or expected value of a random variable \(X\) with probability mass function \(P_X\) is \(E[X] = \sum_k x_k P_X(x_k)\), that is, a weighted average of the values in the range of \(X\).

Exercise 4.13 Consider a random variable \(X\) with \(P_X(x_k) = \frac{1}{7}\) for \(x_k \in \{-3, -2, -1, 0, 1, 2, 3\}\). Let another random variable \(Y = X^2\). Find the range of \(Y\) and the probability mass function \(P_Y\). Using \(P_Y\), find \(E[Y]\).

Theorem 4.2 (Law of the Unconscious Statistician (LOTUS)) Let \(X\) be a random variable, and let \(Y=g(X)\) for some function \(g\). Then we have \(E[Y] = \sum_k g(x_k) P_X(x_k)\). While this holds in general, prove it for the case where the range of \(X\) is finite.

Now we’ll think about measuring the spread or variation of the possible values of \(X\). The measures we’ll use are related to the distance between values of \(X\) and its expectation.

Definition 4.6: Variance and Standard Deviation

Let \(X\) be a random variable with expectation \(E[X]\). Then the variance of \(X\), denoted \(\mathrm{Var}(X)\), is given by \(\mathrm{Var}(X) = E[(X-E[X])^2]\). The square root of variance is called standard deviation, or \(\mathrm{SD}(X)\).

Exercise 4.14 Find \(\mathrm{Var}(X)\) if \(X\) is the outcome when a fair six-sided die is rolled.

The definition of variance isn’t the easiest to actually do calculations with. Luckily, there’s an easier way. Let’s see it and prove that it works:

Theorem 4.3 Prove that \(\mathrm{Var}(X)\) can be written as \(E[X^2] - (E[X])^2\).

Exercise 4.18 Let \(X\) be a random variable, and let \(Y = e^X\). (In function form, this means that \(Y(x) = e^{X(x)}.\)) Find the cumulative distribution function \(F_Y\) in terms of \(F_X\). (In other words, if \(F_Y (y) = F_X (x)\), how are \(y\) and \(x\) related?)

Exercise 4.19 Consider the following game. You roll a fair die once and observe the result \(X\). At this point, you can stop the game and win \(X\) dollars. You can also choose to roll the die for the second time to observe the value \(Y\). In this case, you will win \(Y\) dollars. Let \(W\) be the value that you win in this game. What strategy do you use to maximize \(E[W]\)? What is the maximum \(E[W]\) you can achieve using your strategy? What is \(\mathrm{Var}(W)\) using your strategy?6
